   - `unittest.mock`을 활용해 AI 파이프라인 실행 구간을 Mocking 처리하여, 기능의 동작 흐름은 검증하면서도 테스트가 독립적이고 성공적으로 완료될 수 있도록 리팩토링했습니다.
